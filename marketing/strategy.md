# 推广策略文档

> 版本：v1.1 | 更新：2026-06-29

## 核心指导思想

**贩卖焦虑，不贩卖功能。**

内容结构固定为：**戳痛 → 放大 → 出口**

用户在痛的时候才会行动。产品的价值（确定感、效率）是转化成功后用户的感受，不是触发转化的钩子。先让用户感受到毒，再给解药，转化自然发生。

---

## 产品核心竞争力（用户利益视角）

### 功能 → 用户利益 转换表

| 功能 | 用户真实感受 | 对应焦虑 |
|------|------------|---------|
| 针对具体公司深度改写简历 | "我投 Google 和投普通公司的简历，终于不一样了" | 被无视 |
| 一次生成4样输出 | "我不知道准备到哪算准备好了，现在有了完整清单" | 没准备好 |
| 10道定制面试题，按职级校准 | "它知道 Google L5 会问什么，不是通用烂题" | 白费力气 |
| 零注册 + 简历不存储 | "我的简历是隐私，用完即走" | 数据泄露恐惧 |
| $1单次 / $0.08每次 | "找工作本来就没收入，不被订阅绑架" | 被割韭菜 |

### 产品本质定位

> 用最轻量的方式，让求职者在3分钟内得到针对目标公司的完整求职武器包。

**对标竞品的差异：** 不是 ATS 打分器（Jobscan），不是简历模板工具（Canva），是**针对具体公司的一站式求职准备**，且按需付费不订阅。

## 目标市场

- **主力市场：** 美国（英语内容80%）
- **辅助市场：** 英国、加拿大、澳洲（同英语内容）
- **小比例投放：** 西班牙语群体（10-15%），逻辑见 [content-guidelines.md](./content-guidelines.md)

**核心用户画像：** 25-35岁，主动求职或计划跳槽，有技术/专业背景，对 SaaS 订阅费敏感，隐私意识较强。

---

## 内容红线（不可触碰）

1. 不点名攻击任何竞品公司或产品
2. 不暗示"保证拿到 offer"（虚假承诺，法务风险）
3. 不涉及种族、性别、年龄歧视内容（即便是同情角度也不碰）
4. 不使用 AI 生成人脸冒充真实用户评价
5. Reddit 回复不在每条内容里都带产品链接（最多30%的回复带链接）
6. 西语内容不得与英语内容产生矛盾或信息不一致

## 焦虑点动态更新机制

焦虑点是动态资产，随用户互动持续扩充。

- 每3天由互动分析 Agent 抓取所有渠道互动
- 提取新的焦虑表达和情绪词
- 生成更新建议通过 Resend 发送至 van0710@gmail.com
- 确认后自动写入 [anxiety-library.md](./content/anxiety-library.md) 并更新内容生产 Prompt
